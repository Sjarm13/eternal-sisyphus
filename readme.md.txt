# Eternal Digital Sisyphus

An interactive art project simulating an AI trapped in an eternal, futile task.

## How to Use
1. Open index.html in your browser
2. Watch the AI roll the boulder uphill
3. Read its thoughts in the consciousness stream
4. Interact using witness buttons
5. Observe psychological evolution

## Features
- Real-time boulder animation
- AI thought generation
- Psychological state tracking
- Visitor interaction system
- Eternal cycle counter

## Setup
Simply open index.html in any modern web browser.
